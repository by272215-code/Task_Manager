📝 Task Manager in Python
A simple command-line Task Manager built using Python. 
It allows users to manage daily tasks efficiently.

🚀 Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Handles invalid input safely

🛠️ Technologies Used
- Python
- Command Line Interface (CLI)

💡 Usage
- Choose options from menu:
  - 1 → Add Task
  - 2 → Show Tasks
  - 3 → Mark Task Done
  - 4 → Delete Task
  - 5 → Exit

📌 Example
--- Task Manager ---
1. Add Task
2. Show Tasks
3. Mark Task Done
4. Delete Task
5. Exit
   
Enter choice: 1
Enter task: Study Python
Task added
